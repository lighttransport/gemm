Qwen-Image

Fans out one prompt across the checked backends × the selected precision and renders each generated image + timings. Pick fp8, fp4 (native NVFP4 W4A4 OMMA — CUDA sm_120a), or int4 (Nunchaku W4A16 — HIP); fp4/int4 run only on their capable backend, so picking one generates a single image. Variants come from the server registry.
