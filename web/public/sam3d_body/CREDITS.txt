To extend the gallery, drop additional CC0 images into this
directory. The server's /v1/samples endpoint enumerates the
directory at request time, so new files appear in the web UI on
the next page load (no server restart needed).

Recommended properties:
  - single, full-body person, mostly upright
  - moderate occlusion / clutter is fine — RT-DETR-S handles it
  - 768–1280px on the long edge keeps inference under ~10s
