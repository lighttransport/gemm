Each variant's family (qwen / gemma4) is auto-detected from the GGUF header and routes to the matching C runner; backends that don't support a variant are skipped automatically. Similarity vs. reference
Pick one response as the reference; the server computes token-level agreement (HF tokenizer) and character-level edit distance for every other response. A higher token_match_rate / char_similarity ≈ closer to the reference.
